Lesson 3 Highlights

Highlight #1 Version Control

Version Control is a developing and advancing system that continues to grow stronger as time passes. Version control is a certain system that records changes to a file so different versions can be recalled and used again at a later date.

Highlight #2 Basic GIT Commands

There are many GIT commands that can be used, but there are some that are basic that everyone should know. You can tell a git who you are, create a new respitory, add files, commit, push, or even connect to a remote respitory. These basic git commands are not hard to learn or memorize and can help a user tremendously!

Highlight #3 Connecting GIT to C9

Connecting your GIT to your C9 account is very easy and helps out a lot. There are a small list of steps that you need to follow including typing certain code into your C9 terminal. This is a skill that every programmer should learn!

Highlight #4 Advantages of VSC

Version control has many advantages behind its use, but some advantages are bigger than others. VSC allows you to easily compare files, easily spot differences between two or more files, and helps a developer keep track of which version is being developed at the time.

Highlight #5 How many developers use VSC?

The use of VSC is popular among developers. Version control continues to gain popularity as it continues to advance and become easier and more helpful to use. Developers are coming to realize that version control is an extremely helpful and needful tool.
